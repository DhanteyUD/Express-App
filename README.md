### Deploying Express with Heroku


```
1. cd <dir>

2. git init

3. git add .

4. git commit -m "commit message"

5. curl https://cli-assets.heroku.com/install.sh | sh (Enter device password if present)

6. heroku login

7. heroku create

8. git push heroku master

```
